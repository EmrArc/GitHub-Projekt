Installation (Internet Verbindung für die Nutzung der Anwendung notwendig)

1. Node.js installieren 
2. Node.js in Systemvariabeln hinzufügen
3. npm install im Anwendungsordner ausführen um node_modules zu installieren
4. npm start um die Anwendung im localhost zu starten

Der node_modules beinhaltet sehr viele Daten und wird deswegen nicht mit in der ZIP sein. Die dependencies sind alle in der package-lock.json gelistet.
Darüber weden alle Bibliotheken installiert.

src Ordner is unterteilt
-   Component   : beinhaltet die jeweiligen Komponenten/Code
-   geoData     : beinhaltet die json Datei mit den Daten der Sehenswürdigkeiten
-   static      : beinhaltet die Vektor Bilder
